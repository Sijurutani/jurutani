## 📖 Tentang JuruTani

**JuruTani** merupakan inovasi terbaru yang digagas oleh **Politeknik Pembangunan Pertanian Yogyakarta Magelang**, bertujuan membantu petani setempat dan mendukung pertanian Indonesia.

Platform ini hadir sebagai solusi digital untuk menyediakan informasi terkini seputar pertanian, termasuk:

- 📰 **Berita Pertanian** - Update terbaru seputar dunia pertanian
- 💰 **Harga Pangan** - Informasi harga pangan di berbagai pasar
- 🎓 **Edukasi** - Materi pembelajaran dan kursus pertanian
- 🎥 **Video** - Konten video edukatif tentang pertanian
- 💬 **Konsultasi** - Fitur tanya jawab untuk petani

## 🚀 Menjalankan Development

### Prasyarat

- Node.js 18+ 
- pnpm / npm / yarn
- Supabase account

### Instalasi

```bash
# Clone repository
git clone https://github.com/jurutani/jurutani.git
cd jurutani

# Install dependencies
pnpm install

# Copy environment variables
cp .env.example .env

# Jalankan development server
pnpm dev
```

### Environment Variables

Buat file `.env` dan konfigurasi variabel berikut:

```env
# Supabase
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key

# Google Gemini AI
GEMINI_API_KEY=your_gemini_api_key

# Base URL
NUXT_PUBLIC_BASE_URL=http://localhost:3000
```
